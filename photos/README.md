# Wedding party photos

Drop real photos in this folder to fill the expandable bio cards on the
Wedding Party page. No code changes are needed: each card already points at a
predictable filename, and until a file exists the card shows a labelled
placeholder box naming the photo it expects.

Each person supports up to two photos. Use a `.jpg` named after the person,
with `-2` for the second photo. Portrait orientation (roughly 4:5) looks best.
